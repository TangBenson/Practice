          前置條件

          對於本次Workshop，我們需要幾個前置條件。首先，我們需要 .NET SDK 6.0.

          然後我們需要一些編輯器來做一些正確的C#編碼，你可以使用VSCode，或者如果你已經在你的系統上Visual Studio或JetBrains Rider。

          最後一點是，我們將使用我們的 GraphQL IDE Banana Cake Pop.
